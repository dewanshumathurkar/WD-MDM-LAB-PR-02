# 🎓 Professional Student Profile – HTML Practical

A semantic HTML5 webpage designed to help students create a professional online profile for college placement and recruitment purposes. The webpage presents personal, academic, and contact details in a clean, structured format. 💼

## ✨ Features

- 👤 Professional student introduction
- 🖼️ Profile photograph with accessible `alt` text
- 🎓 Educational qualifications section
- 📚 Academic details displayed using a table
- 🎨 Hobbies and interests list
- 📞 Contact information for recruiters
- 🧭 Navigation links to move between sections
- 🏗️ Semantic HTML5 elements for better structure and accessibility
- 📱 Easy to extend with CSS for responsive design

## 🛠️ Technologies Used

- HTML5
- Semantic HTML Elements

## 📁 Project Structure

```text
professional-profile/
│
├── index.html          # Main webpage
├── profile.png         # Student profile photograph
└── README.md           # Project documentation
```

## 📌 Sections Included

1. **Home** – Introduction and profile photograph  
2. **About Me** – Personal and professional summary  
3. **Education** – Academic qualifications 
4. **Hobbies** – Interests and extracurricular activities  
5. **Contact** – Email address, phone number

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Add your image as `profile.png`.
4. Open `index.html` in a web browser.
5. Replace the sample information with your own details.

## 📝 Customization Guide

Update the following details in `index.html`:

- Your full name
- Profile photograph
- Educational qualifications
- Hobbies and interests
- Email address and phone number

After completing this practical, students will be able to:

- ✅ Build a foundation for a personal portfolio website.
